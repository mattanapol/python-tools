## Usage
Try to find similar image of file 0001.jpg inside folder '/Volumes/CRUCIALSSD'
```
python app.py ./0001.jpg '/Volumes/CRUCIALSSD' -t 95
```
or run with application to prompt input
```
python app.py
```

## Setup
### Create venv with uv
```
uv venv
```

### Activate venv
```
source .venv/bin/activate
```
